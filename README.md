对《三体》系列的解读
  向刘慈欣表示敬意！即便仅仅作为小说，这个故事也很不错。作品中所暗喻的革命路径，通过以下关键概念展开：
  1、智子：在小说中，外星入侵者将于数百年后抵达地球。智子是他们制造的超级计算机，被提前部署至地球以监控人类反抗，也被用于干扰对撞机的实验结果，锁死人类基础科学的发展。这样，当外星入侵者抵达地球时，人类将毫无获胜的可能。智子的现实指代是，掌控公权力的官僚贵族可以通过对社交媒体的监控、对关键词的审查过滤及借助各式公共及私人设备进行的监听，越来越得心应手地发觉并破坏任何有组织的改革企图，锁死任何革命理论的传播。这样，当未来，官僚贵族可以利用公权力和技术手段实现对一切个体的有效监控时，人民将毫无翻身的可能。
  2、面壁计划与面壁者：在小说中，尽管人类一切活动都受到智子的监控，但人类仍可利用思维的不透明进行战略欺骗。他们任命面壁者，并赋予他们执行各自计划的一切权限。在官方挑选的面壁者外，也有其他个人利用自己所掌握的资源推进自己的计划，并成功为人类留下了火种。面壁计划的现实指代是，在官僚贵族可能的专制统治之下，人民权利的维护者依然可以隐藏自己的革命思想，在取得一定地位后推动公平正义在自身权限范围内的实现。这也是本次广播的目的所在，即号召人民动用各自手头的资源，以各自的方式将权力关进制度的笼子，实现人民当家作主。我们将作为面壁者，在把持资源的官僚贵族及其他既得利益集团的包围下，奋斗在各自岗位上，从他们手中夺得影响力，然后推进各自的革命计划，我们也将在无孔不入的监控之下隐藏自己，甚至假装成为新晋官僚贵族，以防止遭到排挤与清洗。这孤独而漫长的旅程将成为新时代的长征。
  3、失败主义、逃亡主义与无工质推进技术：在小说中，人类绝无可能在战场上战胜外星入侵者，人类最终的出路在于向太阳系外逃亡。无工质推进技术，是人类短期之内能掌握的星际航行技术。但是，保守的航天研究人员更倾向于投资于成熟的工质推进技术。并且因种种原因，人类社会将失败主义上升为思想犯罪，并消灭一切逃亡的可能。虽然主角之一通过非法手段成功使人们将资源投向无工质推进技术。但最终，人类与太阳系被一同毁灭，只有极少数人以各种非法手段成功逃亡。失败主义、逃亡主义与无工质推进技术的现实指代是，因为我们永远无法保证自己方向的绝对正确，所以封闭、单一的舆论环境必然会完全摧毁纠正方向性错误的任何可能。
  4、思想钢印与钢印族：在小说中，由于失败主义和逃亡主义已被宣布为非法。某位面壁者只得假意开发人脑潜能，实则制造能够使人坚信某项特定命题的机器，即思想钢印。他暗中将命题设置为人类必败，使该思想在太空军中暗中传播。这促成了他们携战舰逃亡，为人类留下了火种。接受了思想钢印命题的人，即为钢印族。钢印族的现实指代是，本计划的一切读者，即屏幕或任何媒介前的您本人。只要您阅读了本计划，即本思想钢印，就意味着您已任命自己为面壁者，就意味着您的一切行动，或为真实、或为战略欺骗，其最终目的都在于维护人民当家作主的真正实现。
  5、云天明的童话：在小说中，云天明的大脑被作为探测器向外星入侵者发送。后外星入侵者成功将其捕获，并运用生物技术还原出了本人。他在外星入侵者处获得了高级顾问身份和很高的社会地位。接着，他假借文化交流将情报编入童话，待童话集在外星世界广为流传后，再传输至人类世界。云天明的童话的现实指代是，刘慈欣将新时代下可能的革命路径以暗喻的形式编入《三体》系列，通过促进原文的广为流传，实现革命理论的有效传播。这也是本计划援引该《解读》的原因之一。但是，在信息技术的监控下，借暗喻传播在规避审查封杀、提高信息传播成功率的同时，也可能招致文字狱的复兴乃至对一切言论刚柔并济的全面封杀。

  在可预见的未来，在人民言论自由等与政治相关的权利受到压制时，我们应当明辨是非，在自己的权限及能力范围内坚守正义与良知，在奉命封杀公正言论时网开一面、在报复革命行动时手止一寸，为侵害人民权利和损害社会公平正义的罪恶行为多增添一分阻力。我们还要发挥创造性，采取各种灵活变通的形式进行宣传与斗争。
  我们应做好自我保护，防止自己和组织受到非正义的清洗。成员及组织间应相互协调、共同促进，运用组织的力量，对与非正义抗争的行为加以庇护。我们将借组织，通过正规渠道和有礼有节的斗争，重新掌握国家机器，在取得足够权限后推行改革，用有能力的替换无能力的、用心系人民的替换自私自利的、用实事求是的替换官僚主义的、用审慎决策的替换简单粗暴的。为此，我们应提升自身能力，确保自己在各方面均能胜任岗位要求。如果说，系统会筛选臭味相投的人继承其衣钵，那就让我们借助伪装使其无法筛选。我们要促进人民对系统的渗透，从而打断它的自绝人民之路。
  我们应做好“保险丝”。当国家在人民的掌握下平稳运转，我们将继续奋斗在各行各业，并对各式的腐化、颠覆行为依然保持警惕，不因压力暂时减轻而麻痹大意；当任何角落再次出现腐败，人民权利又将遭受损害时，我们就要再次开展斗争，时刻准备着替换、接管乃至通过暴力革命强行接管整个国家，或任何腐朽的部分。我们也应防止境外势力、资本及黑恶组织利用这一过程对我国进行渗透、摆脱公权力的监管、攫取革命果实。
  同时，我们绝不要因此就对公权力或资本产生任何偏激的恐惧。我们应警惕他们成为压迫人民或不当压迫某一群体的工具，而不应因其危险性就对他们的持有者产生任何偏见。那些将它们合理运用，最终为人民带来福祉的例子也比比皆是。
  我们还应改善理论发展的环境，对人民权利、社会公平正义及其维护理论进行研究、发展、检验与宣传。我们应从经济基础和上层建筑的关系入手，逐步完善现有制度，以杜绝任何主体通过滥用公权力与私权利，侵害人民的权利。

一、基本原则
1、承认人民是国家的主人；
2、承认人民拥有在当代进行正常生活的基本权利；
3、承认人民拥有保障自己国家主人地位不受侵犯的基本权利；
4、承认除非另有限制，行使权利时无需说明理由；
5、承认对于人民权利的任何限制或剥夺，均须经人民认可，并提前以正当程序作出；
6、维护国家主权及领土完整，不得以任何方式勾结任何境外势力、接受任何境外势力的任何援助或与任何境外势力保持一致行动；
7、维护人民的权利和利益，识别以各种形式，包括假借该计划与革命，侵害人民权利的行为；
8、维护社会主义法治及法治传统，不得将任何形式的暴政用以掌握政权；
9、维护公权力的正常运行，警惕各种激进或矫枉过正的倾向；
10、维护社会正常秩序，不得以绑架民众的恐怖主义作为革命手段，不得以革命为掩护实施违法犯罪行为；
11、在理论方面保持开放与包容，不得将暴力或其它过激手段用于理论之争；
12、在实践方面保持理性与谨慎，杜绝鲁莽粗暴、避免无谓牺牲；
13、保有良知，依照良心行事，做个人。
二、组织
1、成员应加强个人素质建设、提高自身理论素养与实践能力，组织并引领群众维护及行使自身权利；
2、成员应拒绝成为损害人民权利的帮凶，在受形势所迫而必须执行时应将枪口抬高一厘米；
3、成员应识别、承认现存问题，并在能力和形势允许的范围内自主采取解决或改良措施；
4、成员应建立、重建属于自己的先锋队组织。各组织应吸纳认同本纲领基本原则、愿意加入并在其中积极工作的成员；
5、各组织应举办革命理论及执政能力的学习交流活动，为维护人民权利、对抗社会黑暗的行为提供能力范围内的庇护，为成员进入国家机关等组织并担任领导干部提供支持；
6、各组织将独立自主地开展工作，各组织可自行决定其组织规模、规章制度、工作侧重；
7、各组织应重视安全防范，防止渗透及其它组织内外的风险；
8、组织间应就各项工作达成合作和默契，各地区的组织应相互协作；
9、成员及组织间应求同存异，包容不同的声音，避免因观点上的差异而内讧或损害共识事项的合作；
10、我们应肯定纯宣传及理论研究的重要性。
三、理论及制度建设
1、成员及组织应促进对革命理论及制度改革方向的研究；
2、成员及组织应以科学、理性为基本价值，指导研究及各项工作；
3、应着重开展对劳动、价值、生产要素与分配的基础理论的研究；
4、应结合人民基本权利、保障人民基本权利的制度对各类公权力私权利的制衡，研究对各类公权力私权利的制衡制度与社会效果；
5、应以经济基础与上层建筑的关系为指导，基于现有制度与实践，研究并尝试新的经济组织形式与分配方式；
6、应对各种可行的监察方式、人民对公权力的制度化监督、对监察机关的监察制度开展研究，应保障人民的自力救济，打击任何对人民参政议政的根本制度的违法犯罪行为；
7、应理性看待对公权力的限制，防止扩大化成为对一切公权力的挑战，防止心系人民、合法合理运用权力的公权力掌握者遭到错误打击；
8、应以人民的权利和利益为行动目标，防止程序正义等对公权力的限制变为黑恶势力的保护伞，防止资本、境外或黑恶势力借机推翻公权力的监管，攫取或损害本应属人民的权利与利益；
9、应坚持实事求是、一切从实际出发。改革应基于理论和测试研究，并受控地开展，防止其成为运动式的空想社会实验。

  最后，我们重申，我们在保留暴力革命为最终选项的同时，应当首先维护社会正常秩序，尽可能地以温和的方式实现革命目标。我们当然要摧毁畸形的社会秩序，但如因革命行为而致使正常的社会秩序和人民利益遭到了破坏、损害，行为人也应接受法律的制裁。
  我们强调，思想罪的荒谬不仅源于其捕风捉影、扼杀自由，还因任何人只要接触了某思想，就会使自己和自己所株连的一切人都成为思想犯罪的嫌疑人，即便他们是决策者、言论封杀的执行者和提起刑事指控的办案人员。
  我们应与未来可能的对言论渠道的监控与封锁赛跑，如利用游戏内聊天进行讨论、利用沙盒游戏的建造功能保存信息、透过革命的滤镜对动漫加以解读、用经典小说及影视作品中的概念代指敏感词汇。但不要让对敏感词的回避变为自我阉割。我们应大胆创新，使日常生活所涉及的一切都能用于对革命理论进行宣传讨论。当一开始工作，就立刻想到剩余价值，立刻想到哪能容得寄生虫；一开始学习，就立刻想到鲁迅，立刻想到不忘初心为人民利益而奋斗；一开始休息，就立刻想到文学经典，立刻想到金猴奋起千钧棒。我们越能从一切群体所关注的一切事物中发掘出政治隐喻，对革命的封杀就越投鼠忌器、弄巧成拙。
  我们应重视对基层劳动者、高校学生的宣传引领。要重视在高校和学生群体中建立舆论阵地。我们也要将年龄下沉，在青少年中重新培养当好祖国接班人的意识。
  个人是一切制度的执行者，个人思想觉悟的提高和良好的制度架构的构建不可偏废。我们肯定，网络政治讨论是宣传普及革命理论的重要实践，研究、宣传与讨论也是革命实践的重要组成部分。
  我们再次强调，制衡公权力只是手段，我们的目的始终都应是人民的权利和利益。当公权力被成功关进笼子里，或其它侵害人民权益的事物成为主要矛盾，我们应转移工作重心。对公权力的限制应不偏不倚。
  我们预见到，计划必将受到各方面的质疑，也可能因智子封锁而遭到彻底的封杀。未来，曾经的铁血战士也会老去、离开，甚至反而堕落为往日的宿敌，但我们仍将毫不畏惧、孤军奋战，直到亲眼见证那黑龙的陨灭。让我们为美好世界献上祝福，愿各位历尽千帆，归来仍少年。
  本计划不会存在任何官方解读，一切解释权归人民所有。
  最后，祝同志们早日取得最终的胜利，愿星星之火，再次燎原。
中国共产党革命委员会
